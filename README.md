## Installation
- Clone the repository or download the source code.
  ```bash
  git clone https://github.com/KCoder-programming/html-img-scraper.git
  cd html-img-scraper
  ```

- Install dependencies:
  ```bash
  pip install -r requirements.txt
  ```

## Run the app:
```bash
python download_images.py
```
